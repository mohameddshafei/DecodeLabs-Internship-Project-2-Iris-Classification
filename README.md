# DecodeLabs Internship Project 2 - Iris Classification

## Project Overview

This project is a machine learning classification project completed as part of the DecodeLabs internship.

The project uses the Iris dataset to classify iris flowers into different species based on their measurements. The dataset is loaded directly inside the notebook, so there is no separate CSV file included in this repository.

## Project Objective

The objective of this project is to build a simple machine learning model that can classify iris flowers based on their features.

The model predicts the iris species using measurements such as:

- Sepal length
- Sepal width
- Petal length
- Petal width

## Files Included

- `AI_Project2_Iris_Classification.ipynb` — Jupyter Notebook containing the full project
- `README.md` — Project documentation

## Dataset

The project uses the Iris dataset, which contains flower measurements for three iris species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

No external CSV file is required because the dataset is loaded directly in the notebook.

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Steps

The project follows a basic machine learning workflow:

1. Import the required libraries
2. Load the Iris dataset
3. Explore and inspect the data
4. Separate features and target
5. Split the data into training and testing sets
6. Apply feature scaling
7. Train the K-Nearest Neighbors model
8. Make predictions
9. Evaluate the model performance

## Model Used

- K-Nearest Neighbors Classifier
